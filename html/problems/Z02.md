## 题目描述

计算出$S = 1! + 2! + 3! + ... + n!$. 结果对$998244353$取模. 

其中$!$ 表示阶乘, 定义为$n! = n \times (n-1) \times (n-2) \times \times ... \times 1$. 例如$5! = 5 \times 4 \times 3 \times 2 \times 1 = 120$.

## 输入格式

一行一个正整数$n(1 \le n \le 50)$.

## 输出格式

输出一个整数对$998244353$取模的结果. 

## 输入样例1
```text
5
```

## 输出样例1
```text
153
```

## 输入样例2
```text
20
```

## 输出样例2
```text
696941596
```
